删除 Debian Gnome 所有默认安装的游戏

2022.07.22 10:35 Debian Gnome

我用以下命令删除Debian 11.3（带有gnome）中的所有预装游戏：

    sudo apt purge aisleriot gnome-sudoku mahjongg ace-of-penguins gnomine gbrainy gnome-sushi gnome-taquin gnome-tetravex  gnome-robots gnome-chess lightsoff swell-foop quadrapassel tali && sudo apt autoremove
