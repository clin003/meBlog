上传公钥实现ssh登录

2022.03.12 20:22 linux ssh-copy-id ssh

生成密钥和公钥

如果之前已经存在就不用重新生成了

$ ssh-keygen -t rsa

上传公钥

$ ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.0.1

其中root 服务器用户， 192.168.0.1 是服务器ip。之后按照提示输入密码即可

连接到服务器

直接使用ssh直接连接服务器，无需输入密码

$ ssh root@192.168.0.1
